# Smart Recipe Recommender System

## Overview
The Smart Recipe Recommender System suggests recipes based on ingredients available in a user’s pantry. It also allows users to manage their favorite recipes.

This project was developed as part of an Object-Oriented Analysis & Design (OOAD) course.



## Features
- Add and manage pantry ingredients  
- Generate recipe recommendations  
- View recipe details  
- Add and remove favorite recipes  
- View saved favorites  



## System Design
The system is built using object-oriented principles with the following main components:

- RecipeRecommender – Handles system operations  
- RecommendationService – Calculates match scores and sorts results  
- User – Manages favorites  
- Pantry – Stores ingredients  
- Recipe – Represents recipe data  
- RecommendationResult – Stores recommendation results  



##  Notes
- The system uses rule-based logic (no AI)  
- Focus is on OOAD concepts and UML design
